# CRM Dashboard

A modern Glassmorphism CRM Dashboard built using HTML, CSS, and JavaScript.

## Features
- Add Leads
- Edit Leads
- Delete Leads
- Search Leads
- Filter Leads by Status
- Dashboard Statistics
- Local Storage Support
- Responsive Glassmorphism UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
